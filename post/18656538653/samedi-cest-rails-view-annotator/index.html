Samedi c'est Rails View Annotator

gem install rails_view_annotator

Où est le partial ?

Une question que vous avez déjà pu entendre si vous travaillez avec un graphiste, qui peut avoir du mal à trouver le fichier qui correspond à ce qu’il voit.

Voici donc une gem bien pratique pour ne plus vous répéter. Celle-ci génèrera des commentaires HTML qui engloberont chaque vue.

Par exemple : 

<!-- begin: app/views/user/_bio.html.haml (from app/views/user/show.html.haml:4) -->
<div class='bio'>Ed's Bio</div>
<!-- end: app/views/user/_bio.html.haml (from app/views/user/show.html.haml:4) -->

Une gem à n’utiliser que pendant la phase de développement d’un projet bien entendu :)
