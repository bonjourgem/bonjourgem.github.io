Vendredi c'est Web App Theme

gem install web-app-theme

Cool App Theme

Aujourd’hui une gem bien pratique qui vous donnera une base d’interface propre pour vos applications.

Pour l’utiliser, c’est assez simple. Après avoir ajouté la gem au Gemfile, lancez une petite commande :

rails g web_app_theme:theme

Voilà un layout tout beau tout propre utilisant le thème par défaut. Il y a bien-sûr d’autres thèmes à votre disposition, que vous pouvez utiliser en passant un argument —theme=”my-theme”.

Ensuite générez vos controllers comme d’habitude. Et pour leur donner un peu plus de style, une petite commande :

rails g web_app_theme:themed monmodels

Le thème par défaut ressemble à ça.

Voilà c’est tout pour aujourd’hui :)
