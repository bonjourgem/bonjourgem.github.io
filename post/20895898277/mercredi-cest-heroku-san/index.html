Mercredi c'est Heroku San

gem install heroku_san

Des tâches rake utiles pour Heroku

Aujourd’hui une chouette gem regroupant tout plein de commandes rake qui pourront vous être utiles si vous utilisez Heroku.

Par exemple, pour ajouter le fichier de config d’Heroku à votre application :

rake heroku:create_config

Un exemple plus concret, avec la gem heroku pour effectuer les migrations il faut faire quelque chose dans ce goût-là :

heroku run rake db:migrate

Mais comme on est feignant, avec heroku_san on enlève quelques mots :

rake heroku:db:migrate

Cette gem permet aussi de gérer plus facilement toutes vos applications Heroku via des tâches rake, avec une gestion :

  • Des collaborateurs
  • Du mode maintenance
  • Et d’autres trucs

Intéressé ? Alors bonne lecture !
