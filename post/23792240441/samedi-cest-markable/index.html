Samedi c'est Markable

gem install markable

Une gem qu’elle est bien pour l’utiliser

Aujourd’hui pour bien commencer, une petite vidéo.

Voilà ça c’est fait, maintenant retour au sérieux et place à Markable, une gem drôlement pratique qui permet à vos utilisateurs d’avoir des favoris.

Un petit exemple pour mieux comprendre :

Le Model qui va mettre en favoris

class User > ActiveRecord::Base
  acts_as_marker
end

Un Model qui sera liké / mis en favoris / tout ça tout ça

class Food > ActiveRecord::Base
  markable_as :favorite
end

Et maintenant si on ouvre la commande de notre application :

rails c

On peut tester et admirer les méthodes suivantes : 

# pour mettre en favoris 
user.set_mark :favorite, pizza 
# pour récupérer les favoris
user.favorite_foods

Voilà, c’est tout pour aujourd’hui !
