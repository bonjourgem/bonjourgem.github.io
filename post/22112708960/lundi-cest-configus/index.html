Lundi c'est Configus

gem install configus

La configuration tranquille

Parce que configurer une application peut être ennuyeux, un gentil russe à créé Configus. Cette gem permet d’avoir des configurations qui ressemblent à ceci : 

Configus.build Rails.env do
  env :production do
    website_url 'http://example.com'
    email do
      pop do
        address 'pop.example.com'
        port    110
      end
      smtp do
        address 'smtp.example.com'
        port    25
      end
    end
  end

  env :development, :parent => :production do
    website_url 'http://text.example.com'
    email do
      smtp do
        address 'smpt.text.example.com'
      end
    end
  end
end

Le tout se place dans un initializer.

C’est tout pour ce début de semaine :)
