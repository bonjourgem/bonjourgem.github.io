Vendredi c'est Jackpot

gem install jackpot

Las Vegas Rails app

Aujourd’hui une gem pour gagner plein d’argent ! Enfin, pour implémenter une solution de paiement sur vos applications Rails plutôt :)

Cette gem utilise Active Merchant qui supporte nativement trois plateformes de paiement françaises :

  • CertoDirect
  • Ogone DirectLink
  • PayBox Direct

Une fois la solution de paiement choisie, on configure en créant un initializer : 

Jackpot.configure do |c|
  if Rails.env.production? 
    c.gateway_type      :braintree
    c.gateway_login     ENV['JACKPOT_LOGIN']
    c.gateway_password  ENV['JACKPOT_PASSWORD']
    c.gateway_mode      :production
  else
    c.gateway_type      :braintree
    c.gateway_login     'demo'
    c.gateway_password  'password'
    c.gateway_mode      :test
  end 
  c.default_from      'dont-reply@mycompany.com'
end

Puis il faut aussi ajouter les migrations qui vont bien :

rake jackpot:install:migrations

Et on ajoute Jackpot à nos routes : 

mount Jackpot::Engine => "/billing"

Bon, rien pour Atos, mais peut-être un autre jour :)
