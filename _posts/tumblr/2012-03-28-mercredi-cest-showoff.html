gem install showoff

Powerpoint c’est pour les noobs !

Envie de faire une présentation avec des slides, mais vous êtes allergique à Powerpoint et ses transitions en étoiles ? Pas de souci, voici ShowOff !

Cette gem permet de créer vos présentations avec des fichiers en Markdown et un peu de JSON. Le tout marche avec une application Sinatra. Pour les effets de transition, vous aurez à faire du JavaScript (de nombreux effets étant déjà prêts).

Un petit exemple, à récupérer comme ceci :

git clone https://github.com/schacon/showoff.git

Ensuite on se place dans le répertoire showoff et on lance la commande suivante pour apprécier cette superbe présentation :

showoff serve

On se rend à l’adresse indiquée dans le terminal et là… Mon Dieu que c’est beau ! Vous avez de quoi rigoler un peu pour votre prochaine présentation :)
