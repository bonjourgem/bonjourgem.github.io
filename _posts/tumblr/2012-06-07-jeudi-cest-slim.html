gem install slim

Pour des vues humainement lisibles

Aujourd’hui une gem pour des templates de vues, comme HAML. Voici un petit exemple issu de la documentation : 

doctype html
html
  head
    title Slim Examples
    meta name="keywords" content="template language"

  body
    h1 Markup examples
    #content.example1
      p Nest by indentation

    = yield

    - if items.any?
      table
        - for item in items do
          tr
            td = item.name
            td = item.price
    - else
      p No items found

    #footer
      | Copyright © 2010 Andrew Stone

    = render 'tracking_code'

    script
      | $(content).do_something();

Verdict : c’est aussi rapide qu’HAML. Maintenant à vous de choisir votre moteur de template préféré :)
