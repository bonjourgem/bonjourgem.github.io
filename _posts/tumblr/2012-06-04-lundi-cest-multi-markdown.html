gem install multi_markdown

Des markdowns sans se poser de questions

Aujourd’hui une gem pour utiliser toutes sortes de librairies markdown :

  • BlueCloth
  • kramdown
  • Maruku
  • RDiscount
  • Redcarpet
  • RPEG Markdown
require 'multi_markdown'
MultiMarkdown.new('# Bonjourgem ').to_html

Et hop, la première librairie trouvée sera chargée :)
