gem install rails-mini

Rails et Sinatra ont un bébé

Cette gem vous donne la possibilité de créer des applications simples. Il suffit de modifier un seul et unique fichier, que voici :

Ruby

require 'bundler' 
require 'bundler/setup' 
require 'rails_mini' 
app = Rails.mini do       
    config.secret_token = '!*#&$' * 31       
    get '/' do            
        render text: "Gem Rails like Sinatra. /rails-mini"       
    end       
    get '/hello' do         
        render text: "Hello #{params[:to]}!"     
    end       
    get '/ping' do             
        render json: {'ping' => 'pong'}       
    end 
end 
run app

Mini-application que l’on lance avec la commande suivante :

bundle exec rackup
