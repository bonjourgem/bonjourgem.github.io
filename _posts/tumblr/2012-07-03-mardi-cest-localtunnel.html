gem install localtunnel

Ou comment partager un serveur web local

Vous voulez partager votre application en développement avec un client ou un pote ? Eh bien voici de quoi vous faire plaisir :)

Après avoir installé la gem et lancé votre application, il vous suffit de taper :

localtunnel -k ~/.ssh/id_dsa.pub 3000

Le chemin vers votre clef ssh publique n’est nécessaire que la première fois. En revanche il faut le port à chaque fois !

Quelques secondes plus tard, vous aurez une adresse du type : http://xxxx.localtunnel.com

Voilà, c’est tout pour aujourd’hui !
