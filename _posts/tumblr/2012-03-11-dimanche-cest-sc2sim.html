gem install sc2sim

Pour un build order parfait

Vous êtes totalement taré et passez votre temps à concevoir de meilleures stratégies pour Starcraft 2 ? Alors voici une gem pour occuper votre dimanche.

Il faut la télécharger et l’installer localement comme ceci :

git clone git://github.com/sinisterchipmunk/sc2sim.git

cd sc2sim

gem build sc2sim.gemspec

gem install sc2sim-0.0.0.1.gem

Vous voilà prêt à travailler. Un petit exemple : 

require 'sc2sim'
# on commence la partie en tant que zerg
game = SC2::Simulator.new(:zerg)
# on envoie nos drones bosser 
game.drones.gather :minerals
# on construit un extracteur 
game.build(:extractor)
# etc.

Chaque ordre mettra du temps à être accompli et aura un impact sur vos ressources :)
