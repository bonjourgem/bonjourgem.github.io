gem install middleman

Le site statique tranquille

Besoin de faire un prototype rapidement et de préférence avec du Haml, Sass, CoffeeScript et autres joyeusetés ? Alors dites bonjour à Middleman.

Voici la première commande pour générer un projet :

middleman init mon_projet

Le dossier mon_projet contient un fichier de configuration en Ruby ainsi qu’un dossier sources pour l’Haml, le Sass, les images etc.

Pour faire fonctionner notre application en utilisant les fichiers, une commande :

middleman server

Et enfin pour compiler toutes vos sources et avoir votre petit site statique :

middleman build

Voilà ! C’est tout ce que vous avez besoin de savoir pour faire un site simple avec Middleman.
