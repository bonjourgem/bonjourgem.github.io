gem install pry

L’IRB en mieux

Un IRB avec de jolies couleurs et un explorateur de classe, voilà ce qu’est Pry.

Pour faire l’essai dans votre application Rails, vous pouvez procéder comme ceci :

Gemfile

gem 'pry', :group => :development

environments/development.rb

# à la fin du fichier 
# remplace irb par pry
silence_warnings do 
  require 'pry'
  IRB = Pry
end

Ensuite si vous tapez :

rails console

Vous utiliserez par défaut une console Pry pour faire ce genre de choses :

cd MonModel

Cette commande vous “placera” dans un model afin d’exécuter les méthodes de celui-ci avec des couleurs et tout et tout :

first.unattribut
