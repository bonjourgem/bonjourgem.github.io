gem install recommendable

Mmh, moi aimer !

Il y a deux types de sites : ceux où on peut liker et ceux où on peut pas. Les premiers sont plutôt cool, les seconds moins.

C’est pour cette raison que Recommendable existe, mais aussi parce que l’auteur semble haïr les notations à 5 points.

Pour l’utiliser dans votre prochaine application Rails, quelques étapes :

rails g recommendable:install

Qui génèrera config/initializers/recommendable.rb Vous y trouverez la configuration pour Redis ; dont je parlerai prochainement :)

Ensuite pour permettre à vos visiteurs de liker / disliker, on ajoute la ligne suivante au model utilisateurs :

class User < ActiveRecord::Base
  recommends :bonjourgems
  # ...
end

Vous voilà prêt à faire aimer ou désaimer des trucs divers et variés grâce aux méthodes suivantes :

current_user.like Bonjourgem.find(1)
current_user.dislike Bonjourgem.find(1)
