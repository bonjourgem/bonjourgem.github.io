gem install gog

commit + changelog = Gog ?

Aujourd’hui une gem pour insérer des données pour votre changelog via vos commits.

Finies les mises à jours manuelles de votre log en utilisant ceci dans le message de vos commits : 

# Une ligne
%Bug fix: Fixing issue #2012

# Plusieurs lignes
%%Feature: Adding pluck function to ActiveRecord 
pluck is a function allowing users to select only one column
of a given model.
%%

Maintenant si vous voulez voir un joli changelog, placez-vous dans votre dépôt et tapez :

gog

Tadaam ! Pour la suite, Gog à besoin de vous. N’hésitez pas à faire vos retours à l’auteur.
