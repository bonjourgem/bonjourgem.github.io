gem install rails_autolink

"http://www.bonjourgem.com".to_link

Si vous migrez vers Rails 3.2, cette gem vous permettra de conserver la fonction auto_link présente dans la version 3.1.

Pour les autres qui découvrent cette fonction, elle permet de faire ceci :

Ruby 

auto_link("Visitez donc http://www.bonjourgem.com !")

Ce qui nous produit en sortie :

"Visitez donc <a href=’http://www.bonjourgem.com’>http://www.bonjourgem.com</a>"

Fonction à utiliser par exemple si vous souhaitez remplacer les adresses contenues dans un texte par des liens.
