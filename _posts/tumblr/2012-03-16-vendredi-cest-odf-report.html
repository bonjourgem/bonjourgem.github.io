gem install odf-report

Génère-moi du .odf

Voilà une bien jolie gem pour générer des .odf en utilisant un template .odt que vous aurez créé au préalable.

Par exemple on prépare un template avec des [USER_NAME] un peu partout. Puis on fait un essai avec le code suivant : 

report = ODFReport::Report.new("Chemin/vers/mon_template.odt") do |r|
  r.add_field :user_name, 'Chuck Norris'
end

Tout simplement ! Ça marche aussi avec les images : 

r.add_image :graphics1, "/chemin/vers/mon/image.jpg"

De nombreux exemples d’utilisation sont disponibles sur le dépôt de la gem donc bonne lecture.
