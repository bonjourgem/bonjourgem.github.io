gem install social_stream

Envie de créer votre propre réseau social ? 

La gem Social Stream est basée sur une structure très largement utilisée dans de grosses applications sociales — Facebook entre autres — que l’on appelle SNA.

Pour créer une application Rails sociale, on procède comme d’habitude :

  • Ajout de ‘social_stream’ au Gemfile
  • Petit coup de bundle install

Et on lance le générateur associé pour récupérer les migrations, initializers et autres fichiers :

rails generate social_stream:install

Voilà ! Après avoir effectué les migrations pour votre Base de Données, vous pouvez apprécier votre Facebook fait maison — ou presque ;)
