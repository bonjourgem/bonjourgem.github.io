gem install highgroove_generator

Générez vos projets comme vous le voulez

Aujourd’hui un petit outil bien pratique pour la génération de vos projets, qui marche comme ceci :

highgroove new NomProjet —avec-plein-d-options

Mais quelles options ? Eh bien celle-ci pour commencer :

  • -d pour préciser le type de base de données
  • -h pour avoir une app créée automatiquement sur Heroku
  • -r gestion de version Ruby à utiliser

Et en plus en cadeau dans l’app :

  • Slim
  • RSpec
  • Factory Girl
  • Forgery
  • Capybara-Webkit
  • SimpleCov

Maintenant avec tout ça, vous êtes prêt à commencer une application Rails sur de très bonnes bases :)
