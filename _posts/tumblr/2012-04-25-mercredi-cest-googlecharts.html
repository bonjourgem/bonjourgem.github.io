gem install googlecharts

Pour de belles data visualizations

Envie d’utiliser Google Chart sans difficulté ? Et bien voilà une solution prête à l’emploi.

Un petit exemple issu de la documentation :

require 'googlecharts'
Gchart.line(  :size => '200x300', 
              :title => "example title",
              :bg => 'efefef',
              :legend => ['first data set label', 'second data set label'],
              :data => [10, 30, 120, 45, 72])
