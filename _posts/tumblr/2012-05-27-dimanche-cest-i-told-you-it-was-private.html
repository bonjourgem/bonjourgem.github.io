Aujourd’hui c’est dimanche et comme tous les dimanches voici une gem d’une utilité douteuse :)

Cette petite coquine n’a qu’une seule utilité : on reprend votre code, mais entre-temps vous avez enlevé quelques méthodes ? Ou vous avez juste envie de rigoler ? Alors ça marche comme ceci : 

require 'rubygems'
require 'i-told-you-it-was-private'

class BonjourGem
  i_told_you_it_was_private :gone_method
	
    def another_method
      puts 'uhuh'
    end
end

test = BonjourGem.new
test.another_method
# mais si on essaye 
test.gone_method
# il se passe quelque chose de merveilleux :)

Une surprise vous attend si vous essayez d’utiliser gone_method.

Bon dimanche :)
