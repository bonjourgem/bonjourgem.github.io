gem install git-deploy

Déployer un dépôt git

Aujourd’hui une petite gem pour déployer facilement les derniers changements sur votre dépôt git via FTP ou SSH.

Pour ce faire, rien de plus simple. On commence par placer un deploy.yml à la racine du dépôt :

'example':
  skip: false
  scheme: ftp
  user: example
  password: password
  host: example.com
  port: 21
  path: /path/to/installation
  overwrite_if_same_revision: true

Les différents paramètres sont plutôt clairs, non ? :) Pour lancer le déploiement, taper simplement :

git deploy

Voilà c’est tout pour aujourd’hui.
