gem install meta-tags

Optimise-moi à fond !

En général quand on fait un site, c’est pour qu’il soit vu et utilisé. Il faut donc l’optimiser pour le référencement.

Dans le cas d’une application Rails on peut utiliser MetaTags, une gem comme on les aime avec plein de fonctionnalités :

  • Titre dynamique pour chaque page (géré proprement)
  • De même pour les meta
  • Prise en charge des meta Facebook

Un exemple pour mieux comprendre :

Layout

<head>
  <%= display_meta_tags :site => 'Bonjour gem' %>
</head>

Vue

<h1><%= title 'Jeudi c'est MetaTags' %></h1>

Ce qui nous donnera ce résultat :

<head>
  <title>Bonjour gem | Jeudi c'est MetaTags</title>
</head>
<body>
  <h1>Jeudi c'est MetaTags</h1>
</body>
