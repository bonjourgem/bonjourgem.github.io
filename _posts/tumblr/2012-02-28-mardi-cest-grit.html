gem install grit

Parce que git c’est cool

Il y a donc une gem pour accéder aux dépôts git en lecture / écriture, et le tout en Ruby. Oui Monsieur !

Cette gem s’utilise comme ceci :

repo = Grit::Repo.new("/Mon/depot/local/grit")

Une fois votre objet Repo initialisé, vous aurez tout un tas de méthodes à votre disposition :

  • Les commits
  • Les différents fichiers
  • Les branches
  • Et tous les détails qui accompagnent ces éléments
