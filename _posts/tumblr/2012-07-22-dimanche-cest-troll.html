gem install troll

Et si on trollait un peu ?

Comme le veut la tradition du dimanche, une gem inutile mais amusante :p

Celle-là sert à troller les développeurs Ruby. Lancez simplement :

troll chemin/vers/un/dossier

Et là une nuée de ; apparaitra dans les fichiers Ruby !

Bon malheureusement cela ne marche pas pour tous les fichiers, seulement ceux qui sont dans un répertoire lib/*

Pour tester rapidement faites :

git clone https://github.com/scudco/troll.git

Puis :

troll troll

Voilà :) C’est tout pour aujourd’hui !
