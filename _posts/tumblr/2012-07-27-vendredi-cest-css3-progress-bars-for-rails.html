gem install css3-progress-bar-rails

Twitter Bootstrap 2 Progress Bar

Aujourd’hui une petite gem sans prétention qui donne accès à un helper pour afficher des barres de progression dans vos applications Rails.

L’helper en question s’écrit comme ceci : 

<-- Version standard -->
<%= progress_bar(33, :color => 'blue', :rounded => true) %>  

<-- Version Twitter bootstrap -->
<%= bootstrap_progress_bar(33, :color => 'info', :striped => true) %>

Voilà, c’est tout pour aujourd’hui :)
