gem install ore

Générateur tout en un

Ore est un générateur pour projet Ruby avec quelques options comme :

  • Ajout de Git / Mercurial / SVN
  • Ajouter les tâches de Gem::Tasks/Bundler::GemHelper/Jeweler::Tasks/Gem::PackageTask
  • Ajouter un .rvmrc prêt à l’emploi
  • Ajouter la doc RDoc / YARD
  • Etc.

Contrairement à d’autres générateurs de projet, Ore propose de nombreux templates prêts à l’usage mais aussi la possibilité d’en installer directement depuis un dépôt git !

Le tout marche avec cette simple commande :

mine mon-projet —mes-arguments —tout-ça-tout-ça

Voilà, c’est tout pour aujourd’hui ! :)
