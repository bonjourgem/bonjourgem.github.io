gem install nyan-cat-formatter

Des tests colorés

Marre de voir vos tests tourner sans fun inclus ? Eh bien dites bonjour à cette gem qui vous donnera envie de faire encore plus de tests :)

Pour l’utiliser, on ajoute ceci dans le fichier .rspec :

--format NyanCatFormatter

Ensuite on lance les tests :

rspec

Et on apprécie le résultat !!!

N’oubliez pas d’ajouter la gem dans le Gemfile pour vos projets qui utilisent Bundler :)

C’est tout pour aujourd’hui :)
