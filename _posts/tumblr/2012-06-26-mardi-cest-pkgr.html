gem install pkgr

Transformer vos applications Rails en paquet Debian

Non non ce n’est pas une blague, et cela se fait en quelques minutes :

On ajoute quelques gems au Gemfile

gem 'pkgr'
gem 'thin'

On initialise le dépôt git

cd my-rails-app

git init

git add .

git commit -m “Premier commit”

Pour préparer le pkgr

rake pkgr:setup

Pour générer les fichiers nécessaires à la création du package

rake pkgr:generate

Maintenance c’est prêt à l’emploi. Il n’y a plus qu’à l’installer et à l’utiliser comme n’importe qu’elle paquet Debian :

my-rails-app console development

my-rails-app server start -e development

Voilà, c’est tout pour aujourd’hui :)
