gem install haml

Mhhhh le bon jambon

Haml est au HTML ce que Sass est à la CSS, c’est-à-dire une solution pour générer du HTML avec une syntaxe légère, le tout parfaitement indenté.

Un premier exemple pour vous mettre l’eau à la bouche :

HAML

!!! Strict

HTML

<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">

Et avec des morceaux de Ruby dedans cela donne :

Haml

#main
  .left.column
    #login= current_user.login

HTML

<div id="main">
  <div class="left column">
    <div id="login"><%= current_user.login %></div>
  </div>
</div>

Voilà c’est tout pour aujourd’hui.
