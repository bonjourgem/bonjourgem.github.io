gem install tilt

Une interface générique pour vos moteurs de template

Aujourd’hui c’est Tilt, qui permet d’utiliser n’importe quel type de template de manière générique. Une petite liste de moteurs de template pris en charge :

  • ERB
  • Haml
  • Sass
  • Less CSS
  • RDoc
  • Et bien d’autres…

Maintenant un petit exemple d’utilisation de Tilt :

# on appelle la librairie dont on a besoin, ici erb
require 'erb'
require 'tilt'
template = Tilt.new('templates/foo.erb')
=> #<:erbtemplate>
output = template.render
=> "Hello world!"

Voilà pour la suite ça se passe dans la doc :)
