gem install backbone-on-rails

'Ruby on Rails'.gsub(/Ruby/, 'Backbone')

Si vous ne connaissez pas encore Backbone.js, voici une gem pour commencer sur de bonne bases.

Après l’avoir installée dans votre application Rails, quelques commandes à taper :

rails generate backbone:install

Commande qui vous génèrera la structure d’une application Backbone, il est possible de rajouter —javascript pour ceux qui n’aiment pas CoffeeScript.

rails generate backbone:scaffold monmodel

Génère ce qu’il faut pour manipuler le model avec Backbone.

Pour la suite il va falloir un controller pour le model qui renvoie du JSON et remplir aussi les différents fichiers générés afin d’avoir une application fonctionnelle. Pour cela Ryan Bates a fait un RailsCast sur le sujet, malheureusement payant. Mais les sources sont en accès libre !
