gem install lunchy

Dresseur de démons

Aujourd’hui je vous présente Lunchy, une gem pour contrôler démons, applications et scripts en passant par launchctl, avec une syntaxe plus developer friendly :)

Par exemple, pour voir les services qui peuvent être lancés avec launchctl, on lance la commande suivante :

lunchy ls

Oui, mais lesquels sont en cours ?

lunchy status

Ensuite vous pouvez installer / lancer / stopper / redémarrer tout ce petit monde avec ces commandes :

lunchy install path/to/.plist

lunchy start nomdudémon_ou_delapplication_etc

lunchy stop nomdudémon_ou_delapplication_etc

lunchy restart nomdudémon_ou_delapplication_etc
