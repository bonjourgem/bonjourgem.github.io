Des notifications Growl via Ruby

Vous avez Growl et vous aimeriez avoir des notifications sur des scripts que vous avez faits et qui tournent en fond ? Alors voilà une solution pour vous !

Pour créer une nouvelle notification, ça marche comme ceci : 

require 'ruby_gntp'

growl = GNTP.new("Ruby/GNTP self test")
growl.register({:notifications => [{
  :name     => "notify",
  :title    => "Bonjour gem",
  :text     => "C'est l'heure de faire une pause !",
  :enabled  => true,
}]})

Voilà je vous laisse apprécier le résultat qui est indubitablement de toute beauté :)
