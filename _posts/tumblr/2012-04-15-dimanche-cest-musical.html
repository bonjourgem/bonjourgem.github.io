gem install musical

Give me more mp3

Aujourd’hui c’est dimanche et comme tous les dimanches, voici une gem à l’utilité discutable :) Nous voici donc face à musical, une gem qui permet d’extraire en format mp3 des musiques contenues dans un DVD pour les mettre directement dans iTunes.

Avant toute chose il faut installer quelques pré-requis :

brew install dvdbackup

dvdbackup est un outil en ligne de commande pour riper vos DVD.

brew install ffmpeg

Pour faire de la manipulation audio / vidéo. Il se peut que l’installation avec homebrew ne marche pas. Dans ce cas on l’installe à la main comme ceci :

git clone git://source.ffmpeg.org/ffmpeg.git

cd ffmpeg

./configure —enable-shared —disable-mmx

make install

Voilà maintenant on y est. Il ne reste plus qu’à jouer avec la gem musical, prenez un DVD et essayez la commande suivante :

musical

Drôlement compliqué ! Une liste d’options est à votre disposition : 

# on affiche les données disponibles sur le DVD
--info, -i
# on ripe sans convertir 
--ignore-convert-sound, -g
# on ajoute pas les musiques à iTunes
--ignore-use-itunes, -n
# on règle l'emplacement du DVD
--dev, -d 
# on règle le titre du DVD
--title, -t 
# on règle l'artiste
--artist, -a 
# on indique une nouvelle destination pour les données rippées 
--output, -o 
# la version de musical
--version, -v:

Voilà c’est tout pour aujourd’hui ! Bon dimanche :)
