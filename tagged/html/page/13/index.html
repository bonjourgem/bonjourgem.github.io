gem install formtastic

Le formulaire facile

Mettre en place un formulaire complexe dans une application Rails est un calvaire pour vous ? Formtastic est la gem qu’il vous faut ! Pour construire de beaux formulaires, accessibles et sémantiquement parfaits. 

Elle gère aussi les associations has_many, belongs_to et has_and_belongs_to_many. Cela permet d’afficher des select afin de manipuler les associations entre model plus facilement, depuis les vues.

Et il y a aussi toute une liste de fonctionnalités plutôt cools :

  • I18n
  • Formulaire prêt à être stylé et utilisé avec du JS 
  • Inputs HTML5 (phone, email, search) et les attributs qui vont avec (required, min, max, step, placeholder)
