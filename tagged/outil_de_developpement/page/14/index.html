Créez vos propres gems

Vous avez fait un script que vous pensez utile ? Eh bien il est temps d’en faire une gem ! Pour cela vous pouvez utiliser Jeweler.

Bref, pour commencer on crée un nouveau projet Jeweler :

jeweler ma-gem

Pendant la génération de votre squelette de gem, vous aurez droit à quelques questions comme :

  • Votre nom
  • Votre email
  • Votre compte GitHub
  • Etc.

Maintenant que c’est généré, on peut commencer à travailler en éditant le fichier lib/ma-gem.rb.

Une fois que la gem est prête à être partagée, une seul commande à taper :

rake release

Ce qui aura pour effet de mettre à jour votre dépôt Github et d’envoyer votre gem sur rubygems.org

Voilà, il ne reste plus qu’à faire quelque chose d’utile (ou pas).
