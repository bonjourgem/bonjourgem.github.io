Dimanche c'est Psychedelic

gem install psychedelic

Rhoo y a pleins de jolies couleurs !

Aujourd’hui c’est dimanche et comme à l’accoutumée je vous présente une gem inutile mais amusante.

Voici donc Psychedelic, qui permet d’avoir une console sous acide. Si vous vous souvenez d’Awesome Print, je vous conseille une combinaison des deux pour vous péter la rétine tout en gardant des sorties de scripts lisibles et ordonnées.

Pour l’utiliser, rien de plus simple. Après l’avoir installée, il vous suffit de l’inclure dans votre script : 

require 'psychedelic'

Et voilà ! Je vous laisse apprécier le résultat.
