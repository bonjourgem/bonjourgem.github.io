Mercredi c'est Valium

gem install valium

Soigner votre anxiété

Vous souffrez d’anxiété chronique due à trop d’instanciation d’objets ActiveRecord ? Prenez un peu de cette gem trois fois par jour pendant une semaine et il n’y paraîtra plus.

Plus sérieusement, vous l’aurez peut-être compris, cette gem permet d’accéder aux attributs d’un Model ActiveRecord sans l’instancier, par exemple : 

# un seul attribut 
Post.where(:published => true).value_of :title

# plusieurs 
Employee.where(:title => 'Sr. Monkey Wrangler').values_of :first_name, :last_name, :hired_at

Une gem à utiliser avec modération, seulement dans le cas où vous n’avez pas besoin de tous les trucs cools provenant d’ActiveRecord :)
