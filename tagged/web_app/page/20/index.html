Twitter on Rails

Envie d’intégrer des fonctionnalités de Twitter dans votre prochaine application ? Voici la gem parfaite pour arriver à vos fins.

Après l’avoir installée comme n’importe quelle gem, vous aurez accès à tout plein de classes : 

Twitter::Configuration  Twitter::List       Twitter::Polygon            Twitter::Settings
Twitter::Cursor         Twitter::Metadata   Twitter::RateLimitStatus    Twitter::Size
Twitter::DirectMessage  Twitter::Mention    Twitter::Relationship       Twitter::Status
Twitter::Favorite       Twitter::Photo      Twitter::Reply              Twitter::Suggestion
Twitter::Follow         Twitter::Place      Twitter::Retweet            Twitter::Trend
Twitter::Language       Twitter::Point      Twitter::SavedSearch        Twitter::User

Fonctionnalités que vous pouvez utiliser comme ceci : 

# retourner ma position
Twitter.user("davidmaylis").location
# retourner mon dernier tweet
Twitter.user_timeline("davidmaylis").first.text

Et pleins d’autres bonne choses disponibles dans la documentation de la gem.
